# Ghosh_2207110_CVBuilder

A simple JavaFX application that builds and displays a CV/resume with profile photo, contact details, education, skills, work experience, and projects.

---

## Features

- Load and display a profile image in a **circular frame**
- Show user details: name, phone, email, address
- Display CV sections: Education, Skills, Work Experience, Projects
- Dynamically generate bullet‑list content from comma‑separated strings
- Responsive layout using JavaFX `BorderPane`, `VBox`, and CSS styling
